# Stock Prediction using LSTM and Random Forest

This project predicts stock prices using LSTM for large datasets and Random Forest for smaller ones. 
It also includes interactive Plotly graphs for visualization.

## How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run the model:
   python stock_predictor.py
3. Enter a stock ticker and date range (e.g., AAPL, TSLA).

## Features
- LSTM & Random Forest Models
- Technical indicators (RSI, Moving Averages)
- Plotly 3D Interactive Graphs
